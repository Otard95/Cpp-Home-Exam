# Space Invaders PRO

### Controls

|Key  |Action    |
|:---:|:--------:|
|A    |Move Left |
|D    |Move Right|
|Space|Fire      |
|Esc  |Quit      |

### How to run

> Open `HomeExam.sln` file. Visual Studio 17 will open. Now press that nice green play button.  
Keep console at hand.

### PreRequisites

 - Visual Studio 17
